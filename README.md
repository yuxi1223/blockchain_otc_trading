# 基於智能合約的未上市櫃股票交易平台
- Date：2021/01
- School：National Taiepi University of Technology
- Department：Information and Financial Management
- Menber：蘇昱覡、劉筑芸、謝欣廷、林家民
- Supervising professor：陳育威 教授

## 一、專案概述
本專案旨在開發一個以 **智能合約** 為核心技術的未上市櫃股票交易平台，透過 **去中心化**、**透明化** 的特性，有效解決現行市場面臨的資訊不對稱與監管不足問題，打造一個 **公開**、**可信** 且 **安全** 的交易生態系統。


## 二、系統需求分析方法
為了深入剖析現有市場的運作機制與痛點，本專案採取以下方法：
1. **資料分析**  
   蒐集市場數據，評估未上市櫃股票交易的結構與現況。
2. **訪談法**  
   與相關機構進行訪談，獲取第一手市場資訊。
### 訪談對象及結果概要
| 訪談單位       | 結果摘要                         |
|----------------|---------------------------------|
| **經濟部商業司**  | 獲得市場未上市櫃公司之基本資訊。       |
| **集保中心**      | 了解股東名冊更新機制與股票保管現況。   |
| **證券期貨局**    | 詳細釐清未上市櫃股票的交易流程。       |


## 三、現行市場問題剖析
經過系統性分析與訪談，本專案歸納出未上市櫃股票交易市場存在的 

**三大核心問題**：
### 1. 資訊不透明，市場缺乏信任
- 交易與查詢渠道匱乏，導致資訊流通不暢，交易資訊封閉且公司資訊不對稱。
- 欠缺政府審核機制，容易引發資訊偽造與欺詐問題。

### 2. 股票流通風險高
- 市場存在虛假空殼公司，誘騙投資者進行高風險投資，造成資金損失。

### 3. 交易監管機制不足
- 缺乏政府部門或統一平台的有效監管，無法保障交易雙方的合法權益，交易風險高。


## 四、智能合約解決方案
針對上述問題，本專案提出以 **智能合約** 為基礎的交易平台解決方案，透過以下技術特性實現市場優化：
1. **去中心化機制**  
   消除中間商角色，降低交易過程中的資訊不透明性，提升市場透明度。

2. **不可否認性與數據安全**  
   所有交易記錄皆儲存於區塊鏈，確保數據無法篡改，保障歷史數據的真實性與可信度。

3. **平台化監管與規範化交易**  
   透過智能合約預設交易規則，實現自動化合約執行，降低投資風險並強化市場秩序。


## 五、結論與展望
基於 **智能合約技術** 的未上市櫃股票交易平台，旨在解決 **資訊不透明**、**流通風險** 與 **監管缺失** 等市場痛點，提供一個 **安全**、**公平**、**透明** 的交易環境。

### 展望未來
- 本平台將有助於強化投資者對未上市櫃市場的信任。
- 推動整體市場的健康發展，促進資本市場更高效、更具公信力的運作。
